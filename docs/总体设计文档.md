## 概述

项目名称：书店管理系统  
作者：蒋天杭

本程序要实现一个书店管理系统，支持图书和人员的数据管理。

## 主体逻辑说明
```
解析指令 -> 权限控制 -> 数据处理 -> 数据库操作
```

## 代码文件结构
```
src/database.hpp // 数据库
src/user.hpp // 用户管理
src/book.hpp // 书籍管理
src/context.hpp // 状态控制与权限管理
src/main.cpp // 主文件，包含指令解析
```

```
main -> Context -> Database(user/book)
```
## 功能设计

请概述各模块的功能和功能结构图；
数据库：保存 String（作为 Key） 到 T（模板，作为 value）的映射关系，同个 Key 可以有多条数据，但数据内容不重。

## 数据库设计
用户数据库，以用户名为 key，密码、权限等级为数据内容。

书籍数据库，分别以 ISBN 号、书名、作者、关键字为 key，创建数据库（共 4 个），value 内容均包括 ISBN、书名、作者、关键字（排序以 ISBN 号为序）

## 类、结构体设计

1. Database 类，数据库（`Database<T>`）
    + insert(key, value)
    + delete(key, value)
    + query(key)
    + getall()
2. User 结构体
    + userid: String
    + username: String
    + passwd: String
    + privilege: int
3. Book 结构体
    + ISBN: String
    + name: String
    + author: String
    + keyword: String
    + price: float
    + quantity: int
4. Context 类，用于状态控制与权限管理
